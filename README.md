# Zapchastyny FrontEnd

Проект на Vue 3 з TypeScript, Composition API та Vuetify.

## Технології

- Vue 3
- TypeScript
- Composition API
- Vuetify 3
- Vite

## Встановлення

```bash
npm install
```

## Запуск проекту

```bash
npm run dev
```

## Збірка

```bash
npm run build
```

## Перегляд зібраного проекту

```bash
npm run preview
```
